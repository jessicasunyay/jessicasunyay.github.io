Relational Database IA

May 2023

An independent project completed as part of the Internal Assessment in the IB Computer Science course. The project centers around building a relational database system for APLUS Learning Center (a small business based in Rockville, Maryland) using Microsoft Access, which stores student and parent information, is protected against data loss, and is organized neatly and intuitively. It incorporates SQL, entity-relationship diagrams, user interface, and SDLC.
